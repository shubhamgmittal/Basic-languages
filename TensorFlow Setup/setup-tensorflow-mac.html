TensorFlow Setup Guide for macOS

Required Software

Before setting up TensorFlow, ensure you have the following installed on your macOS system:


  • Python: Version 3.9 or higher (3.11 recommended)

  • Code Editor: Visual Studio Code (recommended)

  • TensorFlow: Latest stable version

  • pip: Python package manager (comes with Python)

  • Virtual Environment: For isolated project environments

  • Homebrew: Package manager for macOS (recommended)

  • Metal Performance Shaders: GPU acceleration for Apple Silicon (M1/M2/M3)

  • Git: For version control (optional but recommended)

Install Homebrew (Recommended)

Homebrew makes it easy to install development tools on macOS.


Installation Steps:

  1. Open Terminal on your Mac

  2. Copy and paste the following command:

    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

  3. Press Enter and follow the on-screen instructions

  4. Verify installation:

    brew --version

Steps of Installation


Step 1: Install Python (if not using Homebrew)


  1. Download Python installer from the official website

  2. Double-click the installer and follow the setup wizard

  3. Accept the license agreement and click "Continue"

  4. Choose the installation destination

  5. Click "Install" and wait for the installation to complete

  6. Verify the installation by opening Terminal and running:

    python3 --version
    pip3 --version

Terminal showing successful Python and pip installation on macOS



Step 2: Create Virtual Environment


  1. Open Terminal on your Mac

  2. Navigate to your project folder:

    cd ~/Documents/Projects

  3. Create a virtual environment:

    python3 -m venv tensorflow-env

  4. Activate the virtual environment:

    source tensorflow-env/bin/activate

  5. You should see (tensorflow-env) prefix in your Terminal prompt

Terminal showing virtual environment activation on macOS



Step 3: Install TensorFlow


  1. Make sure your virtual environment is activated

  2. Upgrade pip to the latest version:

    pip install --upgrade pip

  3. Install TensorFlow (Intel or Apple Silicon):

    pip install tensorflow (Intel Mac)
    pip install tensorflow-macos (Apple Silicon M1/M2/M3)

  4. Wait for installation to complete (may take several minutes)

  5. Verify TensorFlow installation:

    python -c "import tensorflow as tf; print(tf.__version__)"

Terminal showing successful TensorFlow installation on macOS



Step 4: Install Metal Performance Shaders (Apple Silicon Only)


  1. For Apple Silicon (M1/M2/M3) GPU acceleration, install:

    pip install tensorflow-metal

  2. Verify GPU acceleration:

    python -c "import tensorflow as tf; print(tf.config.list_physical_devices('GPU'))"

Step 5: Install Jupyter Notebook (Optional but Recommended)


  1. With virtual environment activated, install Jupyter:

    pip install jupyter

  2. Launch Jupyter Notebook:

    jupyter notebook

  3. A web browser will open with Jupyter Notebook interface

Step 6: Install VS Code


  1. Using Homebrew (recommended):

    brew install --cask visual-studio-code

  2. Or download directly from the official website

  3. Launch VS Code:

    code

Step 7: Install VS Code Extensions


  1. Open Visual Studio Code

  2. Press Cmd+Shift+X to open the Extensions panel

  3. Search for Python and TensorFlow related extensions

  4. Click "Install" for each extension

  5. Reload VS Code by pressing Cmd+Shift+P and selecting "Developer: Reload Window"

Train a Simple Model


import tensorflow as tf
from tensorflow import keras

# Load MNIST dataset
(x_train, y_train), (x_test, y_test) = keras.datasets.mnist.load_data()

# Normalize the data
x_train = x_train.astype('float32') / 255
x_test = x_test.astype('float32') / 255

# Flatten the images
x_train = x_train.reshape(-1, 28*28)
x_test = x_test.reshape(-1, 28*28)

# Create model
model = keras.Sequential([
    keras.layers.Dense(128, activation='relu', input_shape=(784,)),
    keras.layers.Dense(10, activation='softmax')
])

# Compile
model.compile(optimizer='adam', loss='sparse_categorical_crossentropy', metrics=['accuracy'])

# Train
model.fit(x_train, y_train, epochs=5, batch_size=32)

# Evaluate
score = model.evaluate(x_test, y_test, verbose=0)
print(f"Test accuracy: {score[1]*100:.2f}%")

Using Keras Functional API


import tensorflow as tf
from tensorflow import keras
from tensorflow.keras import layers

# Create model using Functional API
inputs = keras.Input(shape=(784,))
x = layers.Dense(128, activation='relu')(inputs)
x = layers.Dropout(0.2)(x)
x = layers.Dense(64, activation='relu')(x)
outputs = layers.Dense(10, activation='softmax')(x)

model = keras.Model(inputs=inputs, outputs=outputs)

# Compile model
model.compile(
    optimizer='adam',
    loss='sparse_categorical_crossentropy',
    metrics=['accuracy']
)

print("Functional API model created!")
model.summary()

Image Classification Example


import tensorflow as tf
from tensorflow import keras

# Load pre-trained MobileNetV2 model
model = keras.applications.MobileNetV2(weights='imagenet')

# Load and preprocess an image
image = keras.preprocessing.image.load_img('image.jpg', target_size=(224, 224))
image_array = keras.preprocessing.image.img_to_array(image)
image_array = keras.applications.mobilenet_v2.preprocess_input(image_array)
image_batch = tf.expand_dims(image_array, 0)

# Make predictions
predictions = model.predict(image_batch)
decoded_predictions = keras.applications.mobilenet_v2.decode_predictions(predictions)[0]

print("Predictions:")
for label, description, score in decoded_predictions:
    print(f"{description}: {score*100:.2f}%")

Troubleshooting Common Issues


Issue 1: Python3 Command Not Found

Problem: 'python3: command not found' error in Terminal

Solution: Install Python via Homebrew: brew install python@3.11


Issue 2: pip Not Found

Problem: 'pip: command not found' or 'pip3: command not found'

Solution: Install with Python: brew install python@3.11 or try python3 -m pip --version


Issue 3: Virtual Environment Not Activating

Problem: Virtual environment fails to activate

Solution: Ensure you're in the correct directory and run source tensorflow-env/bin/activate


Issue 4: TensorFlow Import Error

Problem: ModuleNotFoundError: No module named 'tensorflow'

Solution: Install TensorFlow: pip install tensorflow (Intel) or pip install tensorflow-macos (Apple Silicon)


Issue 5: Apple Silicon Compatibility

Problem: TensorFlow not working on M1/M2/M3

Solution: Use pip install tensorflow-macos and pip install tensorflow-metal for GPU acceleration


Issue 6: GPU Not Detected on Apple Silicon

Problem: Metal GPU acceleration not enabled

Solution: Install Metal plugin: pip install tensorflow-metal and verify with python -c "import tensorflow as tf; print(tf.config.list_physical_devices('GPU'))"


Issue 7: Jupyter Notebook Not Starting

Problem: 'jupyter: command not found'

Solution: Install Jupyter: pip install jupyter in your virtual environment
