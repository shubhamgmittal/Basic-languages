TensorFlow Setup Guide for Linux

Required Software

Before setting up TensorFlow, ensure you have the following installed on your Linux system:


  • Python: Version 3.9 or higher (3.11 recommended)

  • Code Editor: Visual Studio Code (recommended)

  • TensorFlow: Latest stable version

  • pip: Python package manager (comes with Python)

  • Virtual Environment: For isolated project environments

  • Terminal/Shell: Bash or Zsh for command-line operations

  • CUDA & cuDNN: GPU support (optional but recommended)

  • Git: For version control (optional but recommended)

Steps of Installation


Step 1: Update System Packages


  1. Open Terminal on your Linux system

  2. Update package manager:

    sudo apt update && sudo apt upgrade (Debian/Ubuntu)
    sudo dnf update (Fedora/RHEL)
    sudo pacman -Syu (Arch Linux)

Step 2: Install Python and Essential Tools


  1. Install Python 3 and pip:

    sudo apt install python3 python3-pip python3-venv (Debian/Ubuntu)
    sudo dnf install python3 python3-pip (Fedora/RHEL)
    sudo pacman -S python pip (Arch Linux)

  2. Verify the installation:

    python3 --version
    pip3 --version

Terminal showing successful Python and pip installation on Linux



Step 3: Create Virtual Environment


  1. Open Terminal on your Linux system

  2. Navigate to your project folder:

    cd ~/Documents/Projects

  3. Create a virtual environment:

    python3 -m venv tensorflow-env

  4. Activate the virtual environment:

    source tensorflow-env/bin/activate

  5. You should see (tensorflow-env) prefix in your command prompt

Terminal showing virtual environment activation on Linux



Step 4: Install TensorFlow


  1. Make sure your virtual environment is activated

  2. Upgrade pip to the latest version:

    pip install --upgrade pip

  3. Install TensorFlow:

    pip install tensorflow

  4. Wait for installation to complete (may take several minutes)

  5. Verify TensorFlow installation:

    python -c "import tensorflow as tf; print(tf.__version__)"

Terminal showing successful TensorFlow installation on Linux



Step 5: Install Jupyter Notebook (Optional but Recommended)


  1. With virtual environment activated, install Jupyter:

    pip install jupyter

  2. Launch Jupyter Notebook:

    jupyter notebook

  3. A web browser will open with Jupyter Notebook interface

Step 6: Install VS Code


  1. Using package manager:

    sudo apt install code (Debian/Ubuntu)
    sudo dnf install code (Fedora/RHEL)
    sudo pacman -S code (Arch Linux)

  2. Or use Snapcraft (universal):

    sudo snap install code --classic

  3. Launch VS Code:

    code

Step 7: Install VS Code Extensions


  1. Open Visual Studio Code

  2. Press Ctrl+Shift+X to open the Extensions panel

  3. Search for Python and TensorFlow related extensions

  4. Click "Install" for each extension

  5. Reload VS Code by pressing Ctrl+Shift+P and selecting "Developer: Reload Window"

Train a Simple Model


import tensorflow as tf
from tensorflow import keras

# Load MNIST dataset
(x_train, y_train), (x_test, y_test) = keras.datasets.mnist.load_data()

# Normalize the data
x_train = x_train.astype('float32') / 255
x_test = x_test.astype('float32') / 255

# Flatten the images
x_train = x_train.reshape(-1, 28*28)
x_test = x_test.reshape(-1, 28*28)

# Create model
model = keras.Sequential([
    keras.layers.Dense(128, activation='relu', input_shape=(784,)),
    keras.layers.Dense(10, activation='softmax')
])

# Compile
model.compile(optimizer='adam', loss='sparse_categorical_crossentropy', metrics=['accuracy'])

# Train
model.fit(x_train, y_train, epochs=5, batch_size=32)

# Evaluate
score = model.evaluate(x_test, y_test, verbose=0)
print(f"Test accuracy: {score[1]*100:.2f}%")

Using Keras Functional API


import tensorflow as tf
from tensorflow import keras
from tensorflow.keras import layers

# Create model using Functional API
inputs = keras.Input(shape=(784,))
x = layers.Dense(128, activation='relu')(inputs)
x = layers.Dropout(0.2)(x)
x = layers.Dense(64, activation='relu')(x)
outputs = layers.Dense(10, activation='softmax')(x)

model = keras.Model(inputs=inputs, outputs=outputs)

# Compile model
model.compile(
    optimizer='adam',
    loss='sparse_categorical_crossentropy',
    metrics=['accuracy']
)

print("Functional API model created!")
model.summary()

Image Classification Example


import tensorflow as tf
from tensorflow import keras

# Load pre-trained MobileNetV2 model
model = keras.applications.MobileNetV2(weights='imagenet')

# Load and preprocess an image
image = keras.preprocessing.image.load_img('image.jpg', target_size=(224, 224))
image_array = keras.preprocessing.image.img_to_array(image)
image_array = keras.applications.mobilenet_v2.preprocess_input(image_array)
image_batch = tf.expand_dims(image_array, 0)

# Make predictions
predictions = model.predict(image_batch)
decoded_predictions = keras.applications.mobilenet_v2.decode_predictions(predictions)[0]

print("Predictions:")
for label, description, score in decoded_predictions:
    print(f"{description}: {score*100:.2f}%")

Troubleshooting Common Issues


Issue 1: Python3 Command Not Found

Problem: 'python3: command not found' error in Terminal

Solution: Install Python: sudo apt install python3 (Debian/Ubuntu)


Issue 2: pip Not Found

Problem: 'pip: command not found' or 'pip3: command not found'

Solution: Install pip: sudo apt install python3-pip (Debian/Ubuntu)


Issue 3: Virtual Environment Not Activating

Problem: Virtual environment fails to activate

Solution: Ensure you're in the correct directory and run source tensorflow-env/bin/activate (Linux)


Issue 4: TensorFlow Import Error

Problem: ModuleNotFoundError: No module named 'tensorflow'

Solution: Install TensorFlow: pip install tensorflow in your virtual environment


Issue 5: Permission Denied Error

Problem: 'Permission denied' when trying to install packages

Solution: Use virtual environment or add --user flag: pip install --user tensorflow


Issue 6: CUDA/GPU Issues

Problem: TensorFlow not detecting GPU

Solution: Install CUDA and cuDNN, or verify installation with python -c "import tensorflow as tf; print(tf.config.list_physical_devices('GPU'))"


Issue 7: Jupyter Notebook Not Starting

Problem: 'jupyter: command not found'

Solution: Install Jupyter: pip install jupyter in your virtual environment
