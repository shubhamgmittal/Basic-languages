Ruby Setup Guide for Linux

Required Software

Before setting up Ruby, ensure you have the following installed on your Linux system:


  • Linux Distribution: Any modern Linux distribution (Ubuntu, Fedora, Debian, Arch, etc.)

  • Code Editor: Visual Studio Code (recommended)

  • Ruby: Latest stable version (3.x recommended)

  • RubyGems: Ruby package manager (comes with Ruby)

  • Bundler: Dependency manager for Ruby projects

  • Build Tools: gcc, make, and other development tools

  • Git: For version control (optional but recommended)

Download and Install Ruby

Ruby is a dynamic, open-source programming language focused on simplicity and productivity.


  • Method 1: Install via Package Manager (Debian/Ubuntu)
    sudo apt install ruby-full

  • Method 2: Install via Package Manager (Fedora/RHEL)
    sudo dnf install ruby

  • Method 3: Install via Package Manager (Arch Linux)
    sudo pacman -S ruby

  • Method 4: Install via Snapcraft (Universal)
    sudo snap install ruby --classic

  • Method 5: Install via rbenv (Version Manager)
    curl -fsSL https://github.com/rbenv/rbenv-installer/raw/HEAD/bin/rbenv-installer | bash

  • Method 6: Install via RVM (Ruby Version Manager)
    \curl -sSL https://get.rvm.io | bash

Steps of Installation


Step 1: Update System Packages


  1. Open Terminal on your Linux system

  2. Update package manager:

    sudo apt update && sudo apt upgrade (Debian/Ubuntu)
    sudo dnf update (Fedora/RHEL)
    sudo pacman -Syu (Arch Linux)

Step 2: Install Build Tools


  1. Install required build tools:

    sudo apt install build-essential (Debian/Ubuntu)
    sudo dnf groupinstall "Development Tools" (Fedora/RHEL)
    sudo pacman -S base-devel (Arch Linux)

  2. Verify installation:

    gcc --version
    make --version

Step 3: Install Ruby


  1. Open Terminal on your Linux system

  2. Install Ruby using package manager (recommended for beginners):

    sudo apt install ruby-full (Debian/Ubuntu)
    sudo dnf install ruby (Fedora/RHEL)
    sudo pacman -S ruby (Arch Linux)

  3. Verify installation:

    ruby -v
    gem -v

Terminal showing successful Ruby and Gem installation on Linux



Step 4: Install Bundler


  1. Install Bundler:

    sudo gem install bundler

  2. Verify installation:

    bundler -v

Step 5: Create Your First Ruby Project


  1. Create a new directory:

    mkdir hello_ruby

  2. Navigate to the directory:

    cd hello_ruby

  3. Create a Gemfile:

    bundler init

  4. View the project structure:

    ls -la

Step 6: Install VS Code


  1. Using package manager:

    sudo apt install code (Debian/Ubuntu)
    sudo dnf install code (Fedora/RHEL)
    sudo pacman -S code (Arch Linux)

  2. Or use Snapcraft (universal):

    sudo snap install code --classic

  3. Launch VS Code:

    code

Step 7: Install VS Code Extensions


  1. Open Visual Studio Code

  2. Press Ctrl+Shift+X to open the Extensions panel

  3. Search for "Ruby" and install the official Ruby extension

  4. Search for additional Ruby extensions

  5. Click "Install" for each extension

  6. Reload VS Code by pressing Ctrl+Shift+P and selecting "Developer: Reload Window"

Step 8: Run Your First Ruby Program


  1. Open VS Code and create a new file: hello.rb

  2. Add code: puts "Hello, Ruby!"

  3. Save the file

  4. Open integrated terminal in VS Code: Ctrl+`

  5. Run the program:

    ruby hello.rb

  • Ruby Solargraph: Ruby language server

  • Ruby LSP: Language server protocol for Ruby

  • Ruby on Rails: Rails framework support

  • Better Comments: Highlights different comment types

  • GitLens: Git integration for version control

How to Install Extensions


Installation Steps:


  1. Open Visual Studio Code

  2. Press Ctrl+Shift+X to open the Extensions panel

  3. Search for the extension name in the search box

  4. Click the "Install" button next to the extension

  5. Wait for the installation to complete

  6. Click "Reload" if prompted to reload VS Code

Control Flow (if/else)

if age >= 18
  puts "You are an adult"
elsif age >= 13
  puts "You are a teenager"
else
  puts "You are a child"
end

Loops


# for loop
for i in 1..5
  puts "Number: #{i}"
end

# while loop
count = 0
while count < 5
  puts "Count: #{count}"
  count += 1
end

Arrays and Hashes


# Array

puts "First number: #{numbers[0]}"

numbers.each do |num|
  puts "Number: #{num}"
end

puts "Array length: #{numbers.length}"

# Hash

  "name" => "John Doe",
  "age" => 30,
  "email" => "john@example.com"
}

puts "Name: #{person["name"]}"
puts "Age: #{person["age"]}"
puts "Email: #{person["email"]}"

Classes and Objects


class Person
  attr_accessor :name, :age, :email

  def initialize(name, age, email)
    @name = name
    @age = age
    @email = email
  end

  def introduce
    "Hi, I'm #{@name} and I'm #{@age} years old."
  end
end

person = Person.new("John Doe", 30, "john@example.com")
puts person.introduce
puts "Email: #{person.email}"

Error Handling


def divide(a, b)
  begin
    result = a / b
    "Result: #{result}"
  rescue ZeroDivisionError
    "Cannot divide by zero"
  end
end

puts divide(10, 2)
puts divide(10, 0)

Troubleshooting Common Issues


Issue 1: Ruby Command Not Found

Problem: 'ruby: command not found' in terminal

Solution: Ensure Ruby is installed and in PATH, or restart terminal


Issue 2: Gem Command Not Working

Problem: 'gem: command not found'

Solution: Gem comes with Ruby; reinstall Ruby if missing


Issue 3: Permission Denied When Installing Gems

Problem: 'Permission denied' when installing gems

Solution: Use sudo gem install or install to user directory


Issue 4: Bundler Not Found

Problem: 'bundler: command not found'

Solution: Install Bundler: gem install bundler


Issue 5: Ruby Extensions Not Working in VS Code

Problem: No IntelliSense or syntax highlighting

Solution: Install Ruby Solargraph extension and run gem install solargraph


Issue 6: SSL Certificate Issues

Problem: Gem installation fails with SSL errors

Solution: Update Ruby or use gem update --system


Issue 7: Build Tools Missing

Problem: Native extensions fail to compile

Solution: Install build-essential or development tools package

Useful Commands for Ruby Development


  • ruby -v - Check Ruby version

  • gem -v - Check Gem version

  • bundler -v - Check Bundler version

  • ruby file.rb - Run a Ruby file

  • irb - Start interactive Ruby shell

  • gem install gem_name - Install a gem

  • gem list - List installed gems

  • bundle init - Create a new Gemfile

  • bundle install - Install gems from Gemfile

  • bundle exec command - Run command with bundled gems

  • gem update --system - Update RubyGems system

  • gem cleanup - Remove old gem versions
