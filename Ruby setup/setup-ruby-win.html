Ruby Setup Guide for Windows

Required Software

Before setting up Ruby, ensure you have the following installed on your Windows system:


  • Windows: Version 7 or higher (Windows 10/11 recommended)

  • Code Editor: Visual Studio Code (recommended)

  • Ruby: Latest stable version (3.x recommended)

  • RubyGems: Ruby package manager (comes with Ruby)

  • Bundler: Dependency manager for Ruby projects

  • Git for Windows: For version control (optional but recommended)

  • Administrator Access: For installing software

Steps of Installation


Step 1: Install Ruby


  1. Download RubyInstaller from the official website

  2. Run the installer executable (.exe file)

  3. Accept the license agreement and click "Next"

  4. Choose installation directory (default is fine)

  5. Select components to install (include MSYS2 and MINGW if prompted)

  6. Complete the installation

  7. Open Command Prompt and verify installation:

    ruby -v
    gem -v

Command Prompt showing successful Ruby and Gem installation on Windows



Step 2: Install Bundler


  1. Open Command Prompt as Administrator

  2. Install Bundler:

    gem install bundler

  3. Verify installation:

    bundler -v


Step 3: Create Your First Ruby Project


  1. Open Command Prompt

  2. Navigate to your desired location:

    cd Documents

  3. Create a new directory:

    mkdir hello_ruby

  4. Navigate to the directory:

    cd hello_ruby

  5. Create a Gemfile:

    bundler init


Step 4: Install VS Code


  1. Download VS Code installer from the official website

  2. Run the installer executable

  3. Click "Next" and accept the license agreement

  4. Choose installation location (default is fine)

  5. Complete the installation and launch VS Code


Step 5: Install VS Code Extensions


  1. Open Visual Studio Code

  2. Press Ctrl+Shift+X to open the Extensions panel

  3. Search for "Ruby" and install the official Ruby extension

  4. Search for additional Ruby extensions

  5. Click "Install" for each extension

  6. Reload VS Code by pressing Ctrl+Shift+P and selecting "Developer: Reload Window"


Step 6: Run Your First Ruby Program


  1. Open VS Code and create a new file: hello.rb

  2. Add code: puts "Hello, Ruby!"

  3. Save the file

  4. Open integrated terminal in VS Code: Ctrl+`

  5. Run the program:

    ruby hello.rb

How to Install Extensions


Installation Steps:


  1. Open Visual Studio Code

  3. Search for the extension name in the search box

  4. Click the "Install" button next to the extension

  5. Wait for the installation to complete

  6. Click "Reload" if prompted to reload VS Code

  a + b
end

  "Hello, #{name}!"
end

puts "10 + 5 = #{result}"

Control Flow (if/else)


age = 20

if age >= 18
  puts "You are an adult"
elsif age >= 13
  puts "You are a teenager"
else
  puts "You are a child"
end

Loops


# for loop
for i in 1..5
  puts "Number: #{i}"
end

# while loop
count = 0
while count < 5
  puts "Count: #{count}"
  count += 1
end

Arrays and Hashes


# Array
numbers = [1, 2, 3, 4, 5]
numbers.push(6)

puts "First number: #{numbers[0]}"

numbers.each do |num|
  puts "Number: #{num}"
end

puts "Array length: #{numbers.length}"

# Hash
person = {
  "name" => "John Doe",
  "age" => 30,
  "email" => "john@example.com"
}

puts "Name: #{person["name"]}"
puts "Age: #{person["age"]}"
puts "Email: #{person["email"]}"

Classes and Objects


class Person
  attr_accessor :name, :age, :email

  def initialize(name, age, email)
    @name = name
    @age = age
    @email = email
  end

  def introduce
    "Hi, I'm #{@name} and I'm #{@age} years old."
  end
end

person = Person.new("John Doe", 30, "john@example.com")
puts person.introduce
puts "Email: #{person.email}"

Error Handling


def divide(a, b)
  begin
    result = a / b
    "Result: #{result}"
  rescue ZeroDivisionError
    "Cannot divide by zero"
  end
end

puts divide(10, 2)
puts divide(10, 0)

Troubleshooting Common Issues


Issue 1: Ruby Command Not Found

Problem: 'ruby: command not found' in Command Prompt

Solution: Ensure Ruby is added to PATH during installation, or restart Command Prompt


Issue 2: Gem Command Not Working

Problem: 'gem: command not found'

Solution: Gem comes with Ruby; reinstall Ruby if missing


Issue 3: MSYS2/MINGW Not Installed

Problem: Native extensions fail to compile

Solution: Reinstall RubyInstaller with DevKit option


Issue 4: Bundler Not Found

Problem: 'bundler: command not found'

Solution: Install Bundler: gem install bundler


Issue 5: Ruby Extensions Not Working in VS Code

Problem: No IntelliSense or syntax highlighting

Solution: Install Ruby Solargraph extension and run gem install solargraph


Issue 6: SSL Certificate Issues

Problem: Gem installation fails with SSL errors

Solution: Update Ruby or use gem update --system


Issue 7: Permission Denied When Installing Gems

Problem: 'Permission denied' when installing gems

Solution: Run Command Prompt as Administrator or use --user-install flag

Useful Commands for Ruby Development


  • ruby -v - Check Ruby version

  • gem -v - Check Gem version

  • bundler -v - Check Bundler version

  • ruby file.rb - Run a Ruby file

  • irb - Start interactive Ruby shell

  • gem install gem_name - Install a gem

  • gem list - List installed gems

  • bundle init - Create a new Gemfile

  • bundle install - Install gems from Gemfile

  • bundle exec command - Run command with bundled gems

  • gem update --system - Update RubyGems system

  • gem cleanup - Remove old gem versions
