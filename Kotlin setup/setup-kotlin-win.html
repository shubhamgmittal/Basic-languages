Kotlin Setup Guide for Windows

Required Software

Before setting up Kotlin, ensure you have the following installed on your Windows system:


  • Java Development Kit (JDK): Version 8 or higher (Kotlin runs on the JVM)

  • Code Editor: Visual Studio Code or IntelliJ IDEA

  • Kotlin Compiler: Latest stable version

  • Command Line Interface: Windows Command Prompt or PowerShell

Steps of Installation


Step 1: Install Java Development Kit (JDK)


  1. Download JDK from Oracle Official Website

  2. Run the JDK installer

  3. Follow the installation wizard and note the installation path

  4. Set JAVA_HOME environment variable:

    • Right-click "This PC" → Properties

    • Click "Advanced system settings"

    • Click "Environment Variables"

    • Add new System variable: JAVA_HOME = C:\Program Files\Java\jdk-xx

Step 2: Install Kotlin Compiler


  1. Download Kotlin from Kotlin Official Site

  2. Extract the Kotlin zip file to a desired directory

  3. Add Kotlin's bin folder to PATH environment variable

  4. Verify installation by running: kotlinc -version

Step 3: Set Up VS Code


  1. Launch Visual Studio Code

  2. Open Extensions (Ctrl+Shift+X)

  3. Install necessary extensions for Kotlin development

  4. Restart VS Code

Required Extensions


Install the following VS Code extensions for optimal Kotlin development:


  • Kotlin Language: Official Kotlin language support extension

  • Code Runner: Allows quick execution of Kotlin code

  • Debugger for Java: Debugging support for Kotlin (runs on JVM)

  • GitLens: Git integration for version control

  • Prettier: Code formatter for clean code formatting

  • Better Comments: Highlights different comment types

Simple Calculator


fun main() {
    val num1 = 10
    val num2 = 20
    
    val sum = num1 + num2
    val difference = num1 - num2
    val product = num1 * num2
    val quotient = num1 / num2
    
    println("Sum: $sum")
    println("Difference: $difference")
    println("Product: $product")
    println("Quotient: $quotient")
}

Using Variables and Conditionals


fun main() {
    val name = "Kotlin Learner"
    val age = 25
    
    println("Name: $name")
    println("Age: $age")
    
    if (age >= 18) {
        println("You are an adult!")
    } else {
        println("You are a minor!")
    }
}

Simple Loop


fun main() {
    for (i in 1..5) {
        println("Number: $i")
    }
}

GitLens


GitLens is a VS Code extension that allows you to view and manage your Git repositories directly from the editor. It provides a rich Git integration with features like blame annotations, commit history, and more.


To install GitLens, open the Extensions panel in VS Code and search for "GitLens". Click the "Install" button to install the extension.


Once installed, you can use GitLens to view and manage your Git repositories directly from the editor.
