Kotlin Setup Guide for Linux

Required Software

Before setting up Kotlin, ensure you have the following installed on your Linux system:


  • Java Development Kit (JDK): Version 8 or higher (Kotlin runs on the JVM)

  • Code Editor: Visual Studio Code or IntelliJ IDEA

  • Kotlin Compiler: Latest stable version

  • Terminal/Shell: Bash or Zsh for command-line operations

  • Package Manager: apt, yum, or pacman (depending on your Linux distribution)

Download Visual Studio Code

Here is the installer matching your operating system.

  • Linux (.deb)
    Download VS Code for Debian/Ubuntu
  • Linux (.rpm)
    Download VS Code for Fedora/RHEL

Steps of Installation


Step 1: Install Java Development Kit (JDK)


  1. Update package manager:

    sudo apt update (Debian/Ubuntu)
    sudo dnf update (Fedora/RHEL)

  2. Install JDK:

    sudo apt install openjdk-17-jdk (Debian/Ubuntu)
    sudo dnf install java-17-openjdk-devel (Fedora/RHEL)
    sudo pacman -S jdk-openjdk (Arch Linux)

  3. Verify JDK installation:

    java -version
    javac -version

  4. Set JAVA_HOME environment variable:

    Open ~/.bashrc or ~/.zshrc and add:
    export JAVA_HOME=/usr/lib/jvm/java-17-openjdk-amd64
    export PATH=$JAVA_HOME/bin:$PATH
    Then run: source ~/.bashrc

Step 2: Install Kotlin Compiler


  1. Using SDKMAN (Recommended):

    curl -s "https://get.sdkman.io" | bash
    source "$HOME/.sdkman/bin/sdkman-init.sh"
    sdk install kotlin

  2. Or install via package manager:

    sudo apt install kotlin (Debian/Ubuntu)
    sudo dnf install kotlin (Fedora/RHEL)

  3. Verify Kotlin installation:

    kotlinc -version

Step 3: Set Up VS Code


  1. Open Visual Studio Code

  2. Press Ctrl+Shift+X to open the Extensions panel

  3. Install necessary extensions for Kotlin development

  4. Reload VS Code by pressing Ctrl+Shift+P and selecting "Developer: Reload Window"

Required Extensions


Install the following VS Code extensions for optimal Kotlin development:


  • Kotlin Language: Official Kotlin language support extension

  • Code Runner: Allows quick execution of Kotlin code

  • Debugger for Java: Debugging support for Kotlin (runs on JVM)

  • GitLens: Git integration for version control

  • Prettier: Code formatter for clean code formatting

  • Better Comments: Highlights different comment types

How to Install Extensions


Installation Steps:


  1. Open Visual Studio Code

  2. Press Ctrl+Shift+X to open the Extensions panel

  3. Search for the extension name in the search box

  4. Click the "Install" button next to the extension

  5. Wait for the installation to complete

  6. Click "Reload" if prompted to reload VS Code

Basic Lines of Code for Beginners

Here's a simple Hello World program to get you started with Kotlin:


HelloWorld.kt


// Your first Kotlin program
fun main() {
    println("Hello, World!")
}

Simple Calculator


fun main() {
    val num1 = 10
    val num2 = 20
    
    val sum = num1 + num2
    val difference = num1 - num2
    val product = num1 * num2
    val quotient = num1 / num2
    
    println("Sum: $sum")
    println("Difference: $difference")
    println("Product: $product")
    println("Quotient: $quotient")
}

Using Variables and Conditionals


fun main() {
    val name = "Kotlin Learner"
    val age = 25
    
    println("Name: $name")
    println("Age: $age")
    
    if (age >= 18) {
        println("You are an adult!")
    } else {
        println("You are a minor!")
    }
}

Simple Loop


fun main() {
    for (i in 1..5) {
        println("Number: $i")
    }
}

Working with Functions


fun greet(name: String): String {
    return "Hello, $name!"
}

fun main() {
    val message = greet("Linux Developer")
    println(message)
}
