Kotlin Setup Guide for macOS

Required Software

Before setting up Kotlin, ensure you have the following installed on your macOS system:


  • Java Development Kit (JDK): Version 8 or higher (Kotlin runs on the JVM)

  • Code Editor: Visual Studio Code or IntelliJ IDEA

  • Kotlin Compiler: Latest stable version

  • Terminal: macOS Terminal or iTerm2

  • Package Manager: Homebrew (recommended for easy installation)

  • Git: For version control (optional but recommended)

Install Homebrew (Recommended)

Homebrew is the easiest way to install development tools on macOS.


Installation Steps:


  1. Open Terminal on your Mac

  2. Copy and paste the following command:

    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

  3. Press Enter and follow the on-screen instructions

  4. Verify installation:

    brew --version

Steps of Installation


Step 1: Install Java Development Kit (JDK)


  1. Using Homebrew (Recommended):

    brew install openjdk@17

  2. Link JDK to system path:

    sudo ln -sfn /usr/local/opt/openjdk@17/libexec/openjdk.jdk /Library/Java/JavaVirtualMachines/openjdk-17.jdk

  3. Verify JDK installation:

    java -version
    javac -version

  4. Set JAVA_HOME environment variable:

    Open ~/.zshrc (for macOS Catalina or later) or ~/.bash_profile (for older versions) and add:
    export JAVA_HOME=$(/usr/libexec/java_home -v 17)
    export PATH=$JAVA_HOME/bin:$PATH
    Then run: source ~/.zshrc

Terminal showing successful JDK installation and version verification



Step 2: Install Kotlin Compiler


  1. Using Homebrew (Easiest method):

    brew install kotlin

  2. Or using SDKMAN:

    curl -s "https://get.sdkman.io" | bash
    source "$HOME/.sdkman/bin/sdkman-init.sh"
    sdk install kotlin

  3. Verify Kotlin installation:

    kotlinc -version

Terminal displaying successful Kotlin compiler installation



Step 3: Set Up VS Code


  1. Open Visual Studio Code

  2. Press Cmd+Shift+X to open the Extensions panel

  3. Install necessary extensions for Kotlin development

  4. Restart VS Code by pressing Cmd+Shift+P and selecting "Developer: Reload Window"

VS Code Extensions panel ready for Kotlin development

Required Extensions


Install the following VS Code extensions for optimal Kotlin development:


  • Kotlin Language: Official Kotlin language support extension

  • Code Runner: Allows quick execution of Kotlin code

  • Debugger for Java: Debugging support for Kotlin (runs on JVM)

  • GitLens: Git integration for version control

  • Prettier: Code formatter for clean code formatting

  • Better Comments: Highlights different comment types

How to Install Extensions


Installation Steps:


  1. Open Visual Studio Code

  2. Press Cmd+Shift+X to open the Extensions panel

  3. Search for the extension name in the search box

  4. Click the "Install" button next to the extension

  5. Wait for the installation to complete

  6. Click "Reload" if prompted to reload VS Code

Simple Calculator


fun main() {
    val num1 = 10
    val num2 = 20
    
    val sum = num1 + num2
    val difference = num1 - num2
    val product = num1 * num2
    val quotient = num1 / num2

    println("Difference: $difference")

Using Variables and Conditionals


fun main() {
    val name = "Kotlin Developer"
    val age = 25
    
    println("Name: $name")
    println("Age: $age")
    
    if (age >= 18) {
        println("You are an adult!")

Simple Loop


fun main() {
    for (i in 1..5) {

Working with Functions


fun greet(name: String): String {
    return "Hello, $name!"
}

fun main() {
    val message = greet("macOS Developer")
    println(message)
}

Working with Lists


fun main() {
    val fruits = listOf("Apple", "Banana", "Orange", "Mango")
    
    println("Fruits list:")
    for (fruit in fruits) {
        println("- $fruit")
    }
    
    println("\nTotal fruits: ${fruits.size}")
}

Troubleshooting Common Issues


Issue 1: JAVA_HOME Not Set

Problem: Command not found or JAVA_HOME is not recognized

Solution: Make sure you've added the JAVA_HOME export to your .zshrc or .bash_profile file and run source ~/.zshrc


Issue 2: Kotlin Compiler Not Found

Problem: kotlinc: command not found

Solution: Reinstall Kotlin using brew reinstall kotlin or verify Homebrew installation


Issue 3: M1/M2 Mac Compatibility

Problem: Extension or tool not compatible with Apple Silicon

Solution: Download the Apple Silicon version of VS Code and ensure JDK is the ARM64 version


Issue 4: VS Code Extension Not Working

Problem: Kotlin syntax highlighting not working

Solution: Restart VS Code by pressing Cmd+Shift+P and selecting "Developer: Reload Window"
