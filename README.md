# Basic-languages 🌟

A comprehensive web application that provides step-by-step setup guides for programming languages and frameworks across different operating systems.

## 🚀 Overview

Basic-languages is an interactive web application designed to help developers and students easily set up their development environment. Whether you're a beginner taking your first steps in programming or an experienced developer trying a new technology, this app guides you through the entire setup process with platform-specific instructions.

## ✨ Features

- **20+ Languages & Frameworks**: Comprehensive coverage including Python, JavaScript, React, Angular, Django, Flask, and many more
- **Multi-Platform Support**: Detailed setup guides for Windows, macOS, Linux, and ChromeOS
- **Interactive UI**: Beautiful glassmorphism design with animated star background
- **Step-by-Step Instructions**: Clear, actionable guides with code snippets and commands
- **VS Code Integration**: Includes extension recommendations and editor setup
- **Responsive Design**: Works seamlessly across desktop and mobile devices

## 🎯 How It Works

1. **Choose Your Language**: Select from a grid of programming languages and frameworks
2. **Select Your OS**: Pick your operating system (Windows, macOS, Linux, ChromeOS)
3. **Follow the Guide**: Get detailed, platform-specific setup instructions

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection for downloading software and extensions

### Running the Application

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/basic-languages.git
   cd basic-languages
   ```

2. **Open in browser**:
   - Simply open `index.html` in your web browser
   - No server required - it's a static web application

3. **Start exploring**:
   - Click on any programming language icon
   - Select your operating system
   - Follow the step-by-step setup instructions

Each setup guide typically includes:

- **Required Software**: What you need to install
- **Download Links**: Official sources for tools and editors

### Adding a New Language Setup

1. Create a new folder: `[Language] setup/`
2. Add setup files for each OS: `setup-[language]-[os].html`
3. Update `router.js` with routing logic
4. Add the language card to `index.html`
5. Include appropriate icons in the `icons/` folder
