TypeScript Setup Guide for macOS

Required Software

Before setting up TypeScript, ensure you have the following installed on your macOS system:


  • Node.js: Latest LTS version (includes npm)

  • Code Editor: Visual Studio Code (recommended)

  • TypeScript Compiler: Latest stable version

  • npm or Yarn: Package manager for JavaScript/TypeScript

  • Homebrew: Package manager for macOS (optional but recommended)

  • Git: For version control (optional but recommended)

Install Homebrew (Optional but Recommended)

Homebrew makes it easy to install development tools on macOS.


Installation Steps:

  1. Open Terminal on your Mac

  2. Copy and paste the following command:

    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

  3. Press Enter and follow the on-screen instructions

  4. Verify installation:

    brew --version

Steps of Installation


Step 1: Install Node.js (if not using Homebrew)


  1. Download the appropriate Node.js installer for your macOS version

  2. Double-click the installer and follow the setup wizard

  3. Accept the license agreement and click "Continue"

  4. Choose the installation destination

  5. Click "Install" and wait for the installation to complete

  6. Verify the installation by opening Terminal and running:

    node --version
    npm --version

Terminal showing successful Node.js and npm installation verification



Step 2: Install TypeScript Compiler


  1. Open Terminal on your Mac

  2. Install TypeScript globally using npm:

    npm install -g typescript

  3. Verify TypeScript installation:

    tsc --version

  4. Check npm packages installed globally:

    npm list -g

Terminal showing successful TypeScript compiler installation



Step 3: Set Up Your First TypeScript Project


  1. Create a new project directory:

    mkdir my-typescript-project
    cd my-typescript-project

  2. Initialize a new Node.js project:

    npm init -y

  3. Create TypeScript configuration file:

    tsc --init

  4. Create a src folder for TypeScript files:

    mkdir src

Step 4: Install VS Code Extensions


  1. Open Visual Studio Code

  2. Press Cmd+Shift+X to open the Extensions panel

  3. Search for recommended TypeScript extensions

  4. Click "Install" for each extension

  5. Reload VS Code by pressing Cmd+Shift+P and selecting "Developer: Reload Window"

Simple Calculator with Types


// Typed calculator function
function calculate(num1: number, num2: number, operation: string): number {
    switch (operation) {
        case "add":
            return num1 + num2;
        case "subtract":
            return num1 - num2;
        case "multiply":
            return num1 * num2;
        case "divide":
            return num1 / num2;
        default:
            return 0;
    }
}

console.log(calculate(10, 5, "add"));
console.log(calculate(10, 5, "subtract"));
console.log(calculate(10, 5, "multiply"));

Working with Interfaces


// Define an interface for type safety
interface User {
    name: string;
    age: number;
    email: string;
    isActive?: boolean;
}

const user: User = {
    name: "John Doe",
    age: 30,
    email: "john@example.com",
    isActive: true
};

console.log(`User: ${user.name}, Age: ${user.age}`);

Working with Classes


// TypeScript class with type safety
class Person {
    name: string;
    age: number;

    constructor(name: string, age: number) {
        this.name = name;
        this.age = age;
    }

    greet(): void {
        console.log(`Hello, I am ${this.name} and I am ${this.age} years old.`);
    }
}

const person = new Person("Alice", 28);
person.greet();

Working with Arrays

Troubleshooting Common Issues


Issue 1: npm Command Not Found

Problem: 'npm: command not found' error in Terminal

Solution: Reinstall Node.js or use Homebrew: brew install node


Issue 2: TypeScript Compiler Not Found

Problem: 'tsc: command not found' error

Solution: Run npm install -g typescript to install TypeScript globally


Issue 3: VS Code Extension Not Working

Problem: TypeScript syntax highlighting or IntelliSense not working

Solution: Restart VS Code by pressing Cmd+Shift+P and selecting "Developer: Reload Window"


Issue 4: M1/M2 Mac Compatibility

Problem: Extension or tool not compatible with Apple Silicon

Solution: Download the Apple Silicon version of VS Code and Node.js ARM64 version


Issue 5: Module Not Found Error

Problem: Cannot find module error when importing packages

Solution: Run npm install to install dependencies listed in package.json


Issue 6: Permission Denied Error

Problem: Permission denied when trying to access files

Solution: Use appropriate permissions or run with sudo if necessary
