TypeScript Setup Guide for Linux

Required Software

Before setting up TypeScript, ensure you have the following installed on your Linux system:


  • Node.js: Latest LTS version (includes npm)

  • Code Editor: Visual Studio Code (recommended)

  • TypeScript Compiler: Latest stable version

  • npm or Yarn: Package manager for JavaScript/TypeScript

  • Terminal/Shell: Bash or Zsh for command-line operations

  • Package Manager: apt, yum, or pacman (depending on your Linux distribution)

  • Git: For version control (optional but recommended)

Download Node.js

Node.js comes with npm, which is essential for installing TypeScript and managing packages.


  • Using Package Manager (Recommended):
    sudo apt update (Debian/Ubuntu)
    sudo apt install nodejs npm (Debian/Ubuntu)

    sudo dnf install nodejs npm (Fedora/RHEL)

    sudo pacman -S nodejs npm (Arch Linux)

  • Or Download from Official Website:
    Download Node.js LTS from Official Website

  • Using NVM (Node Version Manager):
    curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
    nvm install node

Steps of Installation


Step 1: Update System Packages


  1. Open Terminal on your Linux system

  2. Update package manager:

    sudo apt update (Debian/Ubuntu)
    sudo dnf update (Fedora/RHEL)
    sudo pacman -Syu (Arch Linux)

Step 2: Install Node.js and npm


  1. Install Node.js and npm using your package manager:

    sudo apt install nodejs npm (Debian/Ubuntu)
    sudo dnf install nodejs npm (Fedora/RHEL)
    sudo pacman -S nodejs npm (Arch Linux)

  2. Verify the installation:

    node --version
    npm --version

Terminal showing successful Node.js and npm installation on Linux



Step 3: Install TypeScript Compiler


  1. Open Terminal on your Linux system

  2. Install TypeScript globally using npm:

    npm install -g typescript

  3. Verify TypeScript installation:

    tsc --version

  4. Check npm packages installed globally:

    npm list -g

Terminal showing successful TypeScript compiler installation on Linux



Step 4: Install VS Code


  1. Using package manager:

    sudo apt install code (Debian/Ubuntu)
    sudo dnf install code (Fedora/RHEL)
    sudo pacman -S code (Arch Linux)

  2. Or use Snapcraft (universal):

    sudo snap install code --classic

  3. Launch VS Code:

    code

Step 5: Set Up Your First TypeScript Project


  1. Create a new project directory:

    mkdir my-typescript-project
    cd my-typescript-project

  2. Initialize a new Node.js project:

    npm init -y

  3. Create TypeScript configuration file:

    tsc --init

  4. Create a src folder for TypeScript files:

    mkdir src

Step 6: Install VS Code Extensions


  1. Open Visual Studio Code

  2. Press Ctrl+Shift+X to open the Extensions panel

  3. Search for recommended TypeScript extensions

  4. Click "Install" for each extension

  5. Reload VS Code by pressing Ctrl+Shift+P and selecting "Developer: Reload Window"

Simple Calculator with Types


// Typed calculator function
function calculate(num1: number, num2: number, operation: string): number {
    switch (operation) {
        case "add":
            return num1 + num2;
        case "subtract":
            return num1 - num2;
        case "multiply":
            return num1 * num2;
        case "divide":
            return num1 / num2;
        default:
            return 0;
    }
}

console.log(calculate(10, 5, "add"));
console.log(calculate(10, 5, "subtract"));
console.log(calculate(10, 5, "multiply"));

Working with Interfaces


// Define an interface for type safety
interface User {
    name: string;
    age: number;
    email: string;
    isActive?: boolean;
}

const user: User = {
    name: "John Doe",
    age: 30,
    email: "john@example.com",
    isActive: true
};

console.log(`User: ${user.name}, Age: ${user.age}`);

Working with Classes


// TypeScript class with type safety
class Person {
    name: string;
    age: number;

    constructor(name: string, age: number) {
        this.name = name;
        this.age = age;
    }

    greet(): void {
        console.log(`Hello, I am ${this.name} and I am ${this.age} years old.`);
    }
}

const person = new Person("Alice", 28);
person.greet();

Troubleshooting Common Issues


Issue 1: npm Command Not Found

Problem: 'npm: command not found' error in Terminal

Solution: Reinstall Node.js: sudo apt install nodejs npm (Debian/Ubuntu)


Issue 2: TypeScript Compiler Not Found

Problem: 'tsc: command not found' error

Solution: Run npm install -g typescript to install TypeScript globally


Issue 3: VS Code Extension Not Working

Problem: TypeScript syntax highlighting or IntelliSense not working

Solution: Restart VS Code by pressing Ctrl+Shift+P and selecting "Developer: Reload Window"


Issue 4: Permission Denied Error

Problem: 'Permission denied' when trying to access files or install packages

Solution: Use sudo for system-level operations or fix npm permissions


Issue 5: Module Not Found Error

Problem: Cannot find module error when importing packages

Solution: Run npm install to install dependencies listed in package.json


Issue 6: Port Already in Use

Problem: Error saying a port is already in use when running a development server

Solution: Kill the process using the port: lsof -ti:3000 | xargs kill -9 (replace 3000 with your port)
