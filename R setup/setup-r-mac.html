R Setup Guide for macOS

Required Software


Before setting up R, ensure you have the following installed on your macOS system:


  • R Programming Language: Latest stable version

  • Code Editor: Visual Studio Code or RStudio

  • RStudio: Integrated Development Environment for R (optional but recommended)

  • Terminal: macOS Terminal or iTerm2

  • Package Manager: Homebrew (recommended for easy installation)

  • Git: For version control (optional but recommended)

Steps of Installation


Step 1: Install Homebrew (if not already installed)


  1. Open Terminal on your Mac

  2. Copy and paste the following command:

    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

  3. Press Enter and follow the on-screen instructions

  4. Verify installation:

    brew --version

Step 2: Install R Programming Language


  1. Using Homebrew (Recommended):

    brew install r

  2. Or download from the official website:

    Visit CRAN macOS page and download the appropriate installer for your macOS version

  3. Verify R installation:

    R --version

Step 3: Install RStudio (Optional but Recommended)


  1. Download RStudio from official website

  2. Run the installer and follow the setup wizard

  3. Launch RStudio to verify installation

Step 4: Set Up VS Code for R


  1. Open Visual Studio Code

  2. Press Cmd+Shift+X to open the Extensions panel

  3. Install necessary extensions for R development

  4. Restart VS Code by pressing Cmd+Shift+P and selecting "Developer: Reload Window"

Required Extensions


Install the following VS Code extensions for optimal R development:


  • R: Official R language support extension

  • R LSP Client: Language Server Protocol support for R

  • R Debugger: Debugging support for R scripts

  • GitLens: Git integration for version control

  • Prettier: Code formatter for clean code formatting

  • Better Comments: Highlights different comment types

How to Install Extensions


Installation Steps:


  1. Open Visual Studio Code

  2. Press Cmd+Shift+X to open the Extensions panel

  3. Search for the extension name in the search box

  4. Click the "Install" button next to the extension

  5. Wait for the installation to complete

  6. Click "Reload" if prompted to reload VS Code

Basic Lines of Code for Beginners

Here's a simple Hello World program to get you started with R:


HelloWorld.R


# Your first R program
print("Hello, World!")

Simple Calculator


# Basic arithmetic operations
num1 <- 10
num2 <- 20

sum <- num1 + num2
difference <- num1 - num2
product <- num1 * num2
quotient <- num1 / num2

print(paste("Sum:", sum))
print(paste("Difference:", difference))
print(paste("Product:", product))
print(paste("Quotient:", quotient))

Using Variables and Conditionals


# Variables and if-else statements
name <- "R Learner"
age <- 25

print(paste("Name:", name))
print(paste("Age:", age))

if (age >= 18) {
  print("You are an adult!")
} else {
  print("You are a minor!")
}

Simple Loop


# For loop in R
for (i in 1:5) {
  print(paste("Number:", i))
}

Working with Vectors


# Creating and manipulating vectors
fruits <- c("Apple", "Banana", "Orange", "Mango")

print("Fruits list:")
for (fruit in fruits) {
  print(paste("-", fruit))
}

print(paste("Total fruits:", length(fruits)))

Basic Data Frame


# Creating a simple data frame
students <- data.frame(
  name = c("Alice", "Bob", "Charlie"),
  age = c(20, 22, 21),
  grade = c("A", "B", "A")
)

print("Student Data:")
print(students)
