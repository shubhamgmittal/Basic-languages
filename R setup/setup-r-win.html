R Setup Guide for Windows

Required Software


To set up R development environment on Windows, you'll need the following components:


  • R: The core R programming language and runtime environment.

  • RStudio (Optional but Recommended): An Integrated Development Environment (IDE) for R that makes development easier.

  • Visual Studio Code: A lightweight code editor with excellent R support through extensions.

  • Git (Optional): For version control and collaboration.

Steps of Installation

Step 1: Install R


  1. Visit the CRAN website (Comprehensive R Archive Network)
  2. Click on "base" to download the base R distribution
  3. Download the latest version of R for Windows
  4. Run the installer (.exe file)
  5. Follow the installation wizard and accept the default settings
  6. R will be installed typically at C:\Program Files\R\R-x.x.x\

Step 2: Install RStudio (Recommended)


  1. Visit Posit RStudio Desktop
  2. Download the free version for Windows
  3. Run the installer and follow the wizard
  4. RStudio will automatically detect your R installation
  5. Complete the installation

Step 3: Install R Extensions in VS Code


  1. Open Visual Studio Code
  2. Go to Extensions (Ctrl+Shift+X)
  3. Search for "R" and install the official R extension by REditorSupport
  4. You may also install "R LSP Client" for enhanced language support
  5. Reload VS Code after installation

Step 4: Verify Installation


Open Command Prompt or PowerShell and run:

R --version

You should see the R version information displayed. If not, add R to your PATH environment variable.

Required Extensions for VS Code

Install these essential extensions to enhance your R development experience in VS Code:



1. R (Official Extension by REditorSupport)


  • Provides syntax highlighting and code completion
  • Supports running R code directly from the editor
  • Includes R terminal integration
  • Supports rmarkdown and Shiny development

2. R LSP Client


  • Language Server Protocol support for R
  • Advanced code navigation and refactoring
  • Improved IntelliSense capabilities

3. R Debugger


  • Debugging tools for R code
  • Breakpoint support and variable inspection

4. R Package Manager


  • Manage R packages from VS Code
  • Easy installation and removal of packages

5. Code Runner


  • Quick code execution for various languages including R
  • Run selected lines or entire scripts

Basic Lines of Code for Beginners


Here are some simple R programs to test your setup and get started with R programming:



Example 1: Hello World

# Simple Hello World program
print("Hello, World!")


Example 2: Working with Variables

# Create variables
name <- "John"
age <- 25
height <- 5.9

# Print variables
print(name)
print(age)
print(height)


Example 3: Basic Arithmetic

# Perform basic calculations
a <- 10
b <- 5

addition <- a + b
subtraction <- a - b
multiplication <- a * b
division <- a / b
modulus <- a %% b

print(addition)      # 15
print(subtraction)   # 5
print(multiplication) # 50
print(division)      # 2
print(modulus)       # 0


Example 4: Working with Vectors

# Create vectors
numbers <- c(1, 2, 3, 4, 5)
names <- c("Alice", "Bob", "Charlie")

# Access elements
print(numbers[1])    # 1
print(names[2])      # Bob

# Vector operations
sum(numbers)         # 15
length(names)        # 3


Example 5: Conditional Statements

# If-else statement
score <- 85

if (score >= 80) {
    print("Grade: A")
} else if (score >= 70) {
    print("Grade: B")
} else {
    print("Grade: C")
}


Example 6: Loops

# For loop
for (i in 1:5) {
    print(paste("Iteration", i))
}

# While loop
x <- 1
while (x <= 3) {
    print(paste("Value:", x))
    x <- x + 1
}


Example 7: Functions

# Define a function
greet <- function(name) {
    message <- paste("Hello,", name)
    return(message)
}

# Call the function
greet("Alice")

# Function with multiple parameters
add_numbers <- function(a, b) {
    return(a + b)
}

result <- add_numbers(10, 20)
print(result)  # 30


Example 8: Working with Data Frames

# Create a data frame
students <- data.frame(
    name = c("Alice", "Bob", "Charlie"),
    age = c(20, 21, 22),
    gpa = c(3.8, 3.5, 3.9)
)

# Display data frame
print(students)

# Access columns
print(students$name)

# Access rows
print(students[1,])


Best Extensions for R Development

To maximize your R development experience, here are the recommended extensions and tools:


Essential Extensions:


  • R (by REditorSupport): The official extension providing comprehensive R language support
  • R LSP Client: Language Server Protocol for advanced IDE features
  • Code Runner: Execute code snippets quickly

Productivity Extensions:


  • Rainbow Brackets: Color-code matching brackets for better code readability
  • Bracket Pair Colorizer: Helps track nested brackets easily
  • Prettier: Code formatter for consistent styling
  • GitLens: Git integration and history tracking

Data Science Extensions:


  • Jupyter: Support for R notebooks and interactive development
  • Excel Viewer: View and edit CSV and Excel files directly
  • R Markdown: Support for creating dynamic documents with R code

Theme and UI Extensions:


  • One Dark Pro: Popular dark theme with excellent contrast
  • Dracula Official: Beautiful dark color scheme
  • Icons Extension: Better file icons for visual identification

Installation Tips:


  1. Open VS Code Extensions panel (Ctrl+Shift+X)
  2. Search for each extension by name
  3. Click "Install" to add the extension
  4. Reload VS Code if prompted
