Rust Setup Guide for Linux

Required Software

Before setting up Rust, ensure you have the following installed on your Linux system:


  • Linux Distribution: Any modern Linux distribution (Ubuntu, Fedora, Debian, Arch, etc.)

  • Code Editor: Visual Studio Code (recommended)

  • Rust: Latest stable version

  • Cargo: Rust package manager (comes with Rust)

  • Build Tools: gcc, g++, and make (for compilation)

  • curl or wget: For downloading Rust

  • Git: For version control (optional but recommended)

Install Build Tools

Build tools are required to compile Rust applications on Linux.


Installation Steps (Debian/Ubuntu):

  1. Open Terminal on your Linux system

  2. Update package manager:

    sudo apt update

  3. Install build-essential package:

    sudo apt install build-essential curl

  4. Verify the installation:

    gcc --version
    g++ --version
    make --version

Installation Steps (Fedora/RHEL):

  1. Open Terminal on your Linux system

  2. Update package manager:

    sudo dnf update

  3. Install development tools:

    sudo dnf groupinstall "Development Tools"

  4. Verify the installation:

    gcc --version

Installation Steps (Arch Linux):

  1. Open Terminal on your Linux system

  2. Update package manager:

    sudo pacman -Syu

  3. Install base-devel package:

    sudo pacman -S base-devel

  4. Verify the installation:

    gcc --version

Steps of Installation


Step 1: Update System Packages


  1. Open Terminal on your Linux system

  2. Update package manager:

    sudo apt update && sudo apt upgrade (Debian/Ubuntu)
    sudo dnf update (Fedora/RHEL)
    sudo pacman -Syu (Arch Linux)

Step 2: Install Build Tools


  1. Install required build tools:

    sudo apt install build-essential curl (Debian/Ubuntu)
    sudo dnf groupinstall "Development Tools" (Fedora/RHEL)
    sudo pacman -S base-devel (Arch Linux)

  2. Verify installation:

    gcc --version
    make --version

Step 3: Install Rust


  1. Open Terminal on your Linux system

  2. Install Rust using rustup (recommended):

    curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

  3. Follow the prompts and select "1) Proceed with installation"

  4. Source the cargo environment:

    source $HOME/.cargo/env

  5. Verify installation:

    rustc --version
    cargo --version

Terminal showing successful Rust and Cargo installation on Linux



Step 4: Create Your First Rust Project


  1. Create a new Rust project:

    cargo new hello_world

  2. Navigate to the project:

    cd hello_world

  3. View the project structure:

    ls -la

Step 5: Install VS Code


  1. Using package manager:

    sudo apt install code (Debian/Ubuntu)
    sudo dnf install code (Fedora/RHEL)
    sudo pacman -S code (Arch Linux)

  2. Or use Snapcraft (universal):

    sudo snap install code --classic

  3. Launch VS Code:

    code

Step 6: Install VS Code Extensions


  1. Open Visual Studio Code

  2. Press Ctrl+Shift+X to open the Extensions panel

  3. Search for "rust-analyzer" and install it

  4. Search for additional Rust extensions

  5. Click "Install" for each extension

  6. Reload VS Code by pressing Ctrl+Shift+P and selecting "Developer: Reload Window"

Step 7: Build and Run Your Project


  1. Open your project folder in VS Code

  2. Build the project:

    cargo build

  3. Run the project:

    cargo run

  4. Build for release (optimized):

    cargo build --release

Required Extensions


Install the following VS Code extensions for optimal Rust development:


  • rust-analyzer: Official Rust language server

  • CodeLLDB: Debugger for Rust

  • Rust Syntax: Syntax highlighting for Rust

  • Better TOML: TOML file support for Cargo.toml

  • crates: Dependency manager for Cargo

  • Better Comments: Highlights different comment types

  • GitLens: Git integration for version control

How to Install Extensions


Installation Steps:


  1. Open Visual Studio Code

  2. Press Ctrl+Shift+X to open the Extensions panel

  3. Search for the extension name in the search box

  4. Click the "Install" button next to the extension

  5. Wait for the installation to complete

  6. Click "Reload" if prompted to reload VS Code

Basic Lines of Code for Beginners

Here are simple Rust programs to get you started:


Hello World Program


fn main() {
    println!("Hello, World!");
}

Variables and Data Types


fn main() {
    // Integer
    let age: i32 = 25;
    
    // String
    let name: &str = "Alice";
    
    // Floating-point
    let height: f64 = 5.8;
    
    // Boolean
    let is_active: bool = true;
    
    println!("Name: {}", name);
    println!("Age: {}", age);
    println!("Height: {}", height);
    println!("Active: {}", is_active);
}

Functions


fn add(a: i32, b: i32) -> i32 {
    a + b
}

fn greet(name: &str) -> String {
    format!("Hello, {}!", name)
}

fn main() {
    let result = add(10, 5);
    println!("10 + 5 = {}", result);
    
    let greeting = greet("Rust");
    println!("{}", greeting);
}

Control Flow (if/else)


fn main() {
    let age = 20;
    
    if age >= 18 {
        println!("You are an adult");
    } else if age >= 13 {
        println!("You are a teenager");
    } else {
        println!("You are a child");
    }
}

Loops


fn main() {
    // for loop
    for i in 1..=5 {
        println!("Number: {}", i);
    }
    
    // while loop
    let mut count = 0;
    while count < 5 {
        println!("Count: {}", count);
        count += 1;
    }
}

Vectors and Collections


fn main() {
    // Vector
    let mut numbers: Vec = vec![1, 2, 3, 4, 5];
    
    // Add element
    numbers.push(6);
    
    // Access element
    println!("First number: {}", numbers[0]);
    
    // Iterate
    for num in &numbers {
        println!("Number: {}", num);
    }
    
    println!("Vector length: {}", numbers.len());
}

Structs


struct Person {
    name: String,
    age: u32,
    email: String,
}

fn main() {
    let person = Person {
        name: String::from("John Doe"),
        age: 30,
        email: String::from("john@example.com"),
    };
    
    println!("Name: {}", person.name);
    println!("Age: {}", person.age);
    println!("Email: {}", person.email);
}

Enums and Pattern Matching


enum Color {
    Red,
    Green,
    Blue,
}

fn main() {
    let my_color = Color::Red;
    
    match my_color {
        Color::Red => println!("The color is red"),
        Color::Green => println!("The color is green"),
        Color::Blue => println!("The color is blue"),
    }
}

Error Handling with Result


fn divide(a: i32, b: i32) -> Result {
    if b == 0 {
        Err(String::from("Cannot divide by zero"))
    } else {
        Ok(a / b)
    }
}

fn main() {
    match divide(10, 2) {
        Ok(result) => println!("Result: {}", result),
        Err(error) => println!("Error: {}", error),
    }
}

Troubleshooting Common Issues


Issue 1: Build Tools Not Installed

Problem: 'gcc: command not found' or 'make: command not found'

Solution: Install build tools: sudo apt install build-essential (Debian/Ubuntu) or sudo dnf groupinstall "Development Tools" (Fedora)


Issue 2: Rust Command Not Found

Problem: 'rustc: command not found' or 'cargo: command not found'

Solution: Source the cargo environment: source $HOME/.cargo/env or restart Terminal


Issue 3: Cargo New Not Working

Problem: 'cargo: command not found' when creating new project

Solution: Ensure Rust is properly installed and environment is sourced


Issue 4: Compilation Errors

Problem: Project fails to compile with linker errors

Solution: Install missing libraries or update Rust: rustup update


Issue 5: rust-analyzer Not Working

Problem: IntelliSense not showing suggestions

Solution: Reinstall rust-analyzer or run rustup component add rust-analyzer


Issue 6: CodeLLDB Debugging Not Working

Problem: Debugger not launching or breakpoints not working

Solution: Update CodeLLDB and rebuild the project in debug mode: cargo build


Issue 7: Permission Denied When Installing

Problem: 'Permission denied' error when installing tools

Solution: Use sudo for system-wide installation or install to home directory

  • cargo fmt - Format code with rustfmt

  • cargo clippy - Run Clippy linter

  • rustup update - Update Rust to latest version

  • rustup component add rust-analyzer - Install rust-analyzer component
