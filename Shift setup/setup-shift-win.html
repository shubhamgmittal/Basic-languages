Shift Setup Guide for Windows

Required Software

To start developing with Shift on Windows, make sure you have the following installed:


  • Shift Compiler/Runtime: Download from the official Shift website or repository.

  • Visual Studio Code: A versatile code editor with extension support.

  • Git (Optional): For version control and collaboration.

Steps of Installation


Step 1: Install Shift


Obtain the Shift compiler/distribution from the official source. Run the installer or unzip the package to a directory such as C:\Shift. Add the bin folder to your PATH environment variable:

setx PATH "%PATH%;C:\Shift\bin"


Step 2: Verify Installation


Open Command Prompt and run:

shift --version

You should see the Shift version output. If not, double-check your PATH.

Required Extensions for VS Code

  • Shift Language Support: Provides syntax highlighting and snippets.

  • Code Runner: Execute Shift scripts quickly.

  • GitLens: Helpful for version control integration.

Basic Lines of Code for Beginners

Try these simple Shift examples to ensure everything works:



Hello World

Variables and Arithmetic

let x = 10;
let y = 5;
let sum = x + y;
println(sum);


Conditional

if x > y {
    println("x is greater");
} else {
    println("y is greater or equal");
}

Best Extensions for Shift Development

  • Shift Language Support: Official or community extension for language features.

  • Code Runner: Run snippets without leaving the editor.

  • Bracket Pair Colorizer: Easier bracket matching in complex code.

  • Prettier (if available): Code formatting for consistent style.

Conclusion

You are now ready to build applications with Shift on Windows. Continue exploring the language documentation, write small programs, and gradually build more complex projects. Happy coding!
