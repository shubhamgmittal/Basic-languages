Shift Setup Guide for Linux

Required Software

Ensure the following are installed before starting Shift development:


  • Shift Compiler/Runtime: Download from the official project site or repository.

  • Visual Studio Code: Recommended code editor with extension support.

  • Git (Optional): Useful for version control and collaboration.

  • Package Manager: Your distribution's package manager (apt, yum, dnf, pacman, etc.)

Download Visual Studio Code

VS Code provides excellent editing experience for Shift code. Install it using your preferred method:



Ubuntu/Debian (snap):

sudo snap install code --classic


Ubuntu/Debian (apt):

sudo apt update
sudo apt install code


Fedora/RHEL:

sudo dnf install code


Arch Linux:

sudo pacman -S code


Manual Download:

  1. Visit code.visualstudio.com/download

  2. Download the .deb or .rpm package for your distro

  3. Install with sudo dpkg -i code_*.deb or sudo rpm -i code-*.rpm

Steps of Installation


Step 1: Install Shift


Download the appropriate Linux binary or source from the official site. Extract it to a directory such as /opt/shift and add its bin folder to your PATH:

sudo tar -xzf shift-linux.tar.gz -C /opt
export PATH="$PATH:/opt/shift/bin"

You can add the export line to ~/.bashrc or ~/.profile for persistence.



Step 2: Verify Installation

shift --version

The version information should print; if not, check your PATH settings.

Required Extensions for VS Code

  • Shift Language Support: Syntax highlighting and snippets.

  • Code Runner: Execute Shift scripts directly from the editor.

Basic Lines of Code for Beginners

Here are some introductory Shift examples:



Hello World

println("Hello, Shift!");


Variables and Arithmetic

let a = 20;
let b = 7;
let result = a - b;
println(result);


Conditional Statements

if a > b {
    println("a is larger");
} else {
    println("b is larger or equal");
}


Loop Example

for i in 1..5 {
    println(i);
}

Best Extensions for Shift Development

  • Shift Language Support: Official or community extension with syntax and intellisense.

  • Code Runner: Quickly run code snippets without leaving the editor.

  • Bracket Pair Colorizer: Helpful for reading nested code.

  • Prettier (if supported): Format Shift files consistently.

Conclusion

Your Shift environment on Linux is now ready. Explore the language documentation, try building small projects, and enjoy developing with Shift. Happy coding!
