Java Setup Guide for macOS

Required Software

To work with Java professionally, you will need to install Java Development Kit (JDK) and a code editor like Visual Studio Code. JDK is a software development kit that includes a compiler, debugger, and other tools needed to develop Java applications.

Download JDK

Here is the installer matching your operating system.

  • Mac (64-bit)
    Download JDK for Mac

Required Extensions

After installing VS Code, you should install the following extensions:

  • Java Extension Pack
  • Debugger for Java

First Java Code

To test your setup, create a new file in Visual Studio Code and name it HelloWorld.java. Then, add the following code to the file: 

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, Java!");
    }
}

Save the file and open your terminal. Navigate to the directory where you saved the file and run the following command: 

javac HelloWorld.java && java HelloWorld

You should see the output "Hello, Java!" in your terminal, which means your Java setup is working correctly! Congratulations on setting up your Java development environment on Mac!
