Flask Setup Guide for Mac

About Flask

Flask is a lightweight, micro web framework written in Python. It was created by Armin Ronacher and released in 2010. Flask is designed to be simple and flexible, allowing developers to build web applications and APIs with minimal boilerplate. Because it does not enforce a particular project layout or dependencies, Flask is often described as “micro” – it provides the essentials and lets you choose the rest.


Key features of Flask include:


  • Minimal core: Only the essential components are included, making it easy to learn and extend.

  • Werkzeug & Jinja2: Built on the powerful Werkzeug WSGI toolkit and Jinja2 templating engine.

  • Routing: Flexible URL routing system.

  • Built‑in development server: Simple server for testing and development.

  • Request handling: Easy access to request data and form processing.

  • Blueprints: Modular components for organizing larger applications.

  • Extensions: A rich ecosystem of extensions for databases, authentication, forms, etc.

  • REST support: Frequently used for building RESTful APIs.

Flask is commonly used for small to medium‑sized web applications, prototypes, APIs, and as the basis for more complex frameworks. Its simplicity and extensibility make it a good choice for beginners and experienced Python developers alike.

Following Steps of Flask


Step 1: Install Python

Download and install Python from the official website. Follow the installation instructions provided for macOS.

Step 2: Create a Virtual Environment

Open Terminal and navigate to your project directory. Run the following command to create a virtual environment:


python3 -m venv venv

Activate the virtual environment:


source venv/bin/activate

Step 3: Install Flask


With the virtual environment activated, run the following command to install Flask: 


pip install Flask

Step 4: Verify Installation

To verify that Flask is installed correctly, run the following command:

python3 -c "import flask; print(flask.__version__)"

Step 5: Create a Flask App

Create a new file called app.py and add the following code:


from flask import Flask
app = Flask(__name__)
@app.route('/')
def hello():
    return "Hello, World!"

Run the Flask app using the following command:


python3 app.py

Open your web browser and navigate to http://localhost:5000
