Flask Setup Guide for Windows

Required Software

  • Python (version 3.6 or higher)

Download Visual Studio Code

Here is the installer matching your operating system.

  • Windows (64-bit)
    Download VS Code for Windows

Steps to Install Flask

Follow the steps below to set up Flask on your Windows machine:



Step 1: Install Python

Download and install Python from the official website. Make sure to check the box that says "Add Python to PATH" during installation.

Step 2: Create a Virtual Environment

Open Command Prompt and navigate to your project directory. Run the following command to create a virtual environment:


python -m venv venv



Activate the virtual environment:


venv\Scripts\activate

Step 3: Install Flask

With the virtual environment activated, run the following command to install Flask:


pip install Flask

Step 4: Verify Installation

To verify that Flask is installed correctly, run the following command:


python -c "import flask; print(flask.__version__)"

Step 5: Create a Flask App


Create a new file called app.py and add the following code:


from flask import Flask
app = Flask(__name__)   
@app.route('/') 
def hello():
    return "Hello, World!"

Run the Flask app using the following command:



python app.py

Open your web browser and navigate to http://localhost:5000

Required Extensions

To enhance your Flask development experience, consider installing the following Visual Studio Code extensions:


  • Python Extension for Visual Studio Code

  • Pylance Extension for Visual Studio Code

Conclusion

You have successfully set up Flask on your Windows machine. You can now start building your web applications using Flask!
