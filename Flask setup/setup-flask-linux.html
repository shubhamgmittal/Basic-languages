Flask Setup Guide for Linux

Required Software

  • Python (version 3.6 or higher)

Download Visual Studio Code

Here is the installer matching your operating system.

  • Windows (64-bit)
    Download VS Code for Windows

Following Steps of Flask


Step 1: Install Python

Most Linux distributions come with Python pre-installed. You can check if Python is installed by running the following command in the terminal:


python3 --version

If Python is not installed, you can install it using your distribution's package manager. For example, on Debian-based systems, you can run:


sudo apt-get install python3

Step 2: Create a Virtual Environment

Open Terminal and navigate to your project directory. Run the following command to create a virtual environment:


python3 -m venv venv

Activate the virtual environment:


source venv/bin/activate

Step 3: Install Flask

With the virtual environment activated, run the following command to install Flask:


pip install Flask

Step 4: Verify Installation

To verify that Flask is installed correctly, run the following command:


python3 -c "import flask; print(flask.__version__)"

Step 5: Create a Flask App

Create a new file called app.py and add the following code:


from flask import Flask
app = Flask(__name__)
@app.route('/')
def hello():
    return "Hello, World!"

Run the Flask app using the following command:


python3 app.py

Open your web browser and navigate to http://localhost:5000


Required Extensions for VS Code

To enhance your development experience in Visual Studio Code, install the following extensions:


  • Python (by Microsoft)

  • Flask Snippets (by William H.)

Conclusion

You have successfully set up Flask on your Linux system. You can now start building your Flask applications!
