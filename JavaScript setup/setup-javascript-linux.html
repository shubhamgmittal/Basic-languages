JavaScript Setup Guide for Linux

Required Software

To work with JavaScript professionally, you will need to install Node.js and a code editor like Visual Studio Code. Node.js is a JavaScript runtime that allows you to run JavaScript code outside of a web browser, and Visual Studio Code is a great code editor that provides excellent support for JavaScript development.

  • JavaScript (ES6) code snippets
  • Auto Rename Tag
  • Bracket Pair Colorizer

Beginner JavaScript Basics

Here is a simple JavaScript line of code that you can run in your terminal: 

console.log("Hello, World!");

Running Your JavaScript Code

To run your JavaScript code, follow these steps:

  1. Open your terminal.
  2. Navigate to the directory where your JavaScript file is located using the cd command.
  3. Run your JavaScript file using the command node yourfile.js, replacing yourfile.js with the name of your JavaScript file.

Expected Output

After running the command, you should see the following output in your terminal: 

Hello, World!
