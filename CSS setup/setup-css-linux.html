CSS Setup Guide for Linux

Required Software

Before setting up CSS development environment, ensure you have the following installed on your Linux system:


  • Linux Distribution: Any modern Linux distribution (Ubuntu, Fedora, Debian, Arch, etc.)

  • Code Editor: Visual Studio Code (recommended)

  • Web Browser: Chrome, Firefox, Chromium, or other modern browser

  • Node.js: Optional but recommended for CSS preprocessors and development tools

  • npm: Comes with Node.js for package management

  • Git: For version control (optional but recommended)

Download Visual Studio Code

Here is the installer matching your operating system.

  • Linux (.deb)
    Download VS Code for Debian/Ubuntu
  • Linux (.rpm)
    Download VS Code for Fedora/RHEL

Steps of Installations


Step 1: Install VS Code


  1. Using package manager:

    sudo apt install code (Debian/Ubuntu)
    sudo dnf install code (Fedora/RHEL)
    sudo pacman -S code (Arch Linux)

  2. Or use Snapcraft (universal):

    sudo snap install code --classic

  3. Launch VS Code:

    code


Step 2: Install a Web Browser (if not already installed)


  1. For Firefox (usually pre-installed):

    sudo apt install firefox (Debian/Ubuntu)
    sudo dnf install firefox (Fedora/RHEL)

  2. For Chromium:

    sudo apt install chromium (Debian/Ubuntu)
    sudo dnf install chromium (Fedora/RHEL)

  3. Launch the browser to verify installation


Step 3: Install Node.js (Optional but Recommended)


  1. Install Node.js using package manager:

    sudo apt install nodejs npm (Debian/Ubuntu)
    sudo dnf install nodejs npm (Fedora/RHEL)
    sudo pacman -S nodejs npm (Arch Linux)

  2. Or from NodeSource repository (for latest version):

    curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
    sudo apt-get install -y nodejs

  3. Verify installation:

    node --version
    npm --version


Step 4: Configure VS Code for CSS Development


  1. Open VS Code

  2. Create a new folder for your project

  3. Open the folder in VS Code: File > Open Folder

  4. Create a new file: index.html

  5. Create a new file: styles.css


Step 5: Install VS Code Extensions for CSS


  1. Open Visual Studio Code

  2. Press Ctrl+Shift+X to open the Extensions panel

  3. Search for CSS-related extensions

  4. Click "Install" for each recommended extension

  5. Reload VS Code by pressing Ctrl+Shift+P and selecting "Developer: Reload Window"


Step 6: Create Your First CSS Project


  1. Create a new folder on your computer:

    mkdir my-css-project
    cd my-css-project

  2. Open it in VS Code

  3. Create index.html file with basic HTML structure

  4. Create styles.css file with CSS rules

  5. Link CSS to HTML: <link rel="stylesheet" href="styles.css">

  6. Open index.html in a web browser

Basic Lines of Code for Beginners Sections

Here are simple CSS examples to get you started:


1. Basic HTML with CSS


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">

    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <h1>Hello, CSS!</h1>
    <p>This is a paragraph.</p>
</body>
</html>

.container {
    width: 80%;
    margin: 0 auto;
    padding: 20px;

    border-radius: 8px;
}

.box {
    margin: 10px;

    border: 1px solid #ddd;

4. Flexbox Layout

    display: flex;
    justify-content: space-between;
    align-items: center;
    padding: 10px;
    background-color: #333;
}

.nav-items {
    display: flex;
    gap: 20px;
}

    display: flex;
    flex-wrap: wrap;
    gap: 15px;
}

5. Grid Layout


.grid-container {
    display: grid;
    grid-template-columns: 1fr 1fr 1fr;
    gap: 20px;
    padding: 20px;
}

.grid-item {
    background-color: #e0e0e0;
    padding: 20px;
    border-radius: 5px;
}

6. Hover Effects and Transitions


button {
    background-color: #3498db;
    color: white;
    padding: 10px 20px;
    border: none;
    border-radius: 5px;
    cursor: pointer;
    transition: background-color 0.3s ease;
}

button:hover {
    background-color: #2980b9;
}

.link {
    text-decoration: none;
    color: #3498db;
    transition: color 0.3s ease;
}

.link:hover {
    color: #2980b9;
}

7. Responsive Design with Media Queries


/* Desktop styles */
.container {
    width: 1200px;
    margin: 0 auto;
}

.columns {
    display: grid;
    grid-template-columns: 1fr 1fr 1fr;
}

/* Tablet styles */
@media (max-width: 768px) {
    .container {
        width: 90%;
    }
    
    .columns {
        grid-template-columns: 1fr 1fr;
    }
}

/* Mobile styles */
@media (max-width: 480px) {
    .container {
        width: 100%;
    }
    
    .columns {
        grid-template-columns: 1fr;
    }
}

How to Install Extensions


Installation Steps:


  1. Open Visual Studio Code

Troubleshooting Common Issues


Issue 1: CSS Not Loading in Browser

Problem: CSS file is not being applied to HTML

Solution: Check the file path in the link tag and ensure styles.css is in the correct directory


Issue 2: Live Server Not Working

Problem: Live Server extension doesn't start or port is in use

Solution: Change the port in settings or close other applications using the port


Issue 3: Extensions Not Showing Syntax Highlighting

Problem: CSS syntax highlighting is not working

Solution: Ensure file has .css extension and reload VS Code


Issue 4: Color Picker Not Appearing

Problem: Color picker doesn't show in CSS files

Solution: Install CSS Peek extension and ensure it's enabled


Issue 5: Browser Caching Issues

Problem: Changes in CSS don't appear in browser

Solution: Hard refresh browser with Ctrl+Shift+R or clear cache


Issue 6: VS Code Won't Install Extensions

Problem: Extension installation hangs or fails

Solution: Check internet connection, restart VS Code, or use command line installation
